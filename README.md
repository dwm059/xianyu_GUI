# xianyu_GUI
闲鱼商品搜索监控加秒拍，windows 端

## 运行界面

监听按钮可以监控最新发布消息，可以根据关键字监控
![img_1.png](img_1.png)
![img_2.png](img_2.png)

闲鱼扫描二维码可以跳转商品链接

秒拍按钮可以直接拍下对应商品（需要账号登录）

可以设置地区以及关键字过滤商品，打开秒拍监控在设置的价格区间内实现自动拍下
![img_3.png](img_3.png)
![img_4.png](img_4.png)

## 使用
使用该产品需配合模拟器（真机）登录闲鱼账号，不登录无法获取最新信息
